# 1992 Suzuki Cappuccino (EA11R)

This repository contains information, code, and design files related to my 1992 Suzuki Cappuccino.

## About the Suzuki Cappuccino

The Suzuki Cappuccino (EA11R) is a small 2-seater detachable hardtop miniature sports car produced by Suzuki. It was designed to meet Japanese Kei car specifications and features a turbocharged 657cc inline-3 engine.

## Repository Contents

This repository serves as a central location for:

- Technical documentation and specifications
- Code for custom modifications and electronics
- CAD files and design drawings
- Maintenance records and guides
- Parts information and resources

## Vehicle Information

- **Model:** Suzuki Cappuccino
- **Chassis Code:** EA11R
- **Year:** 1992
- **Engine:** F6A Turbocharged I3
- **Displacement:** 657cc
